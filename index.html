📍 آدرس: کابل، ناحیه نهم، قریه خواجه رواش، مایار طبی لابراتوار

📋 منوی تخصصی معاینات و آزمایش‌ها:

Blood Chemistry (کیمیای خون)
  • FBS / R.B.S (قند خون)
  • Cholesterol / Triglyceride
  • HDL / LDL (چربی خون)
  • BUN / Urea / Creatinine
  • Bilirubin (Total / Direct)
  • SGPT (ALT) / SGOT (AST)
  • Alkaline Phosphatase
  • Uric Acid / Calcium
Hematology (خون‌شناسی)
  • TLC / RBC / Platelet
  • HCT / HB / ESR
  • Lymphocytes / Monocytes
  • Neutrophils / Eosinophil
  • Basophils / Reticulocytes
  • Malaria (MP) / Leishmania
Urine & Stool (ادرار و مدفوع)
  • Urine Chemical (PH/Sugar)
  • Urine Microscopic (WBC/RBC)
  • Stool Color & Parasites
  • H. Pylori (Stool Test)
Semen Analysis (تجزیه اسپرم)
  • Volume / PH / Liquefaction
  • Sperm Count / Sperm Motility
  • Morphology (Normal/Abnormal)
Serology (سیرولوژی)
  • Widal Test / TyphoidDot
  • Brucellosis (Abortus/Melitensis)
  • Toxoplasmosis (IgG/IgM)
  • Rheumatoid Factor / C.R.P / ASO
  • HBs Ag / HCV Ab / HIV / Blood Group
  • TORCH Test (Toxo, Rubella, CMV)
